🛠 PETUNJUK INSTALASI
Ekstrak file project
Buat database
Import file SQL di folder _db
Atur nama database di file .env
Jalankan terminal pada folder project

Menjalankan server:

php artisan serve

Agar bisa diakses di jaringan LAN:

php artisan serve --host=0.0.0.0 --port=8000

Akses browser:
